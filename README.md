# Farnoush Daliran Portfolio

Personal portfolio for Farnoush Daliran, a full-stack developer based in Munich.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Lucide React

## Sections

- Home
- About
- Projects
- Skills
- Contact links

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

The portfolio highlights selected full-stack projects, core skills, and contact links for job applications.
